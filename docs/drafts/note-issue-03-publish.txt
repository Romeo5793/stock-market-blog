【第3号の公開手順（A: 有料記事）】

所要: 15分
ファイル:
- docs/drafts/note-issue-03-free.txt  → 無料
- docs/drafts/note-issue-03-paid.txt  → 有料 980円

========================
1. 先に有料記事を作る
========================
1. note「投稿」→ 新規投稿
2. note-issue-03-paid.txt を貼る
3. 公開設定で「有料」を選ぶ
4. 価格: 980
5. 無料で読める範囲:
   タイトル直下〜「この記事で読めるもの」まで
6. 公開
7. マガジン「株価調査メモ（週次）」に追加
8. URLをコピー

========================
2. 無料ダイジェストを出す
========================
1. 新規投稿
2. note-issue-03-free.txt を貼る
3. 「→ 公開後、ここに有料記事URLを貼ってください」
   を、さっきの有料URLに書き換える
4. 無料で公開
5. 同じマガジンに追加

========================
3. 相互リンク
========================
有料記事末尾の
「無料ダイジェスト: （公開後にURLを相互リンク）」
も、無料記事URLに更新すると親切です。

========================
4. 終わったら共有
========================
無料URL と 有料URL をチャットへ。
ブログCTAも有料記事へ向けて更新します。
